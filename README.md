# What is ?
GvasViewer  
[Unreal Engine](https://www.unrealengine.com/) Gvas Savedata Format File Viewer

# Advance
* Windows 10(or 11)
* [.NET 9](https://dotnet.microsoft.com/en-us/download/dotnet/9.0)

# Usage
* Prepare GVAS files
* File -> Open
* Using the Tree View
